# Wallpaper Manager

Unified wallpaper manager for VS Code, Cursor, IDEA, and PyCharm.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Run

```bash
python -m wallpaper_manager
```

Expected output: `Wallpaper Manager scaffold OK`

## Tests

```bash
pytest
```
